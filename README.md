--Objetivo geral do Projeto--

O projeto tem como função desenvolver um sistema de marcação carga horaria de trabalho, onde os funcionários batem o ponto, o tempo de inicio de expediente é então marcado e, caso haja tempo de pausa ou encerramento do expediente, o funcionário novamente bate o ponto, mas desta vez isto vai marcar o fim daquele segmento de tempo trabalhado.

OBS: o projeto atual tem como função principal servir de portifolio e forma de treinar programação.
